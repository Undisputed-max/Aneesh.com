Mobile Action RPG Weapon Systems Inventory Shader VFX

A mobile action RPG with weapon variety, real-time combat, and distinct visual identity across character classes. The game required performant shader work for realistic visuals on mobile hardware — a direct constraint that shaped every rendering decision.

My focus: weapon and inventory systems built cleanly enough to support rapid content expansion, shader-based visual enhancements that held up on mid-range Android devices, and mobile performance optimisation that kept frame time stable during heavy combat sequences.

// My responsibilities
  • Designed and implemented a weapon system — weapon components attached to character, swap logic, animation notify integration, hit detection using line/sweep traces with damage type dispatch.
  • Built a modular inventory system — item data assets, equipment slots, pickup/drop flow, and inventory UI bound to gameplay state.
  • Wrote custom material shaders in Unreal's material graph for rim lighting, surface wetness, and emissive weapon effects — all optimised for mobile GPU instruction budgets.
  • Profiled and optimised mobile performance — draw call batching, texture streaming, LOD configuration, and animation tick rate management to hit 60fps on target devices.
// System architecture
Weapons
Component-based Weapon System
UActorComponent subclass per weapon type, attached to character skeleton socket. Fire/reload logic encapsulated per component. Damage dispatched via UGameplayStatics with custom damage type assets for elemental effects.
Inventory
Data-Asset Inventory
Items defined as UDataAsset blueprints — type, rarity, mesh, stats. Inventory component holds TArray of item references. UI widget bound via UMG property binding, updating reactively on inventory mutation.
Rendering
Mobile Shader Optimisation
Custom material functions for rim light, emissive pulse, and surface variation. Instruction counts kept under mobile GPU limits using texture atlases and conditional static switches. Shader complexity visualiser used throughout development.
